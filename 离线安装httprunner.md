# 离线安装httprunner

## 安装好Python3.x环境
使用``` py -V ```查看python安装情况
使用``` pip -V ```查看pip版本情况情况

## 离线安装支持包

### httprunner需要安装以下支持包
- requests
- Py-YAML
- Jinja2
- har2case
- colorama
- colorlog
- requests_toolbelt
在有网络的环境下，通过``` pip install packagename ``` 安装好以上支持包

## 安装httprunner
py setup.py install
